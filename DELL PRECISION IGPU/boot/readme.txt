Dell E6540 UEFI eGPU configurator
=================================

bootx64.efi is a custom grub2 build with a setup_var command to alter UEFI variables.
grub.cfg implements the Dell E6540 UEFI eGPU configurator
grub.cfg can be created/modified for other systems

Install on USB      - copy files into \EFI\boot on USB stick and UEFI boot the USB stick
Install on UEFI ESP - run 'install_on_ESP.bat' as an admin user and follow instructions given
